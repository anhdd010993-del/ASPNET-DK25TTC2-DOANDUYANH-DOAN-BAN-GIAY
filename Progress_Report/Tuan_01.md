# BÁO CÁO TIẾN ĐỘ ĐỒ ÁN - TUẦN 1

- **Khoảng thời gian:** 22/06/2026 - 28/06/2026
- **Trọng tâm công việc:** Giai đoạn chuẩn bị và nghiên cứu lý thuyết cơ sở

## 1. Nội dung công việc
- Xác định lý do chọn đề tài "Xây dựng hệ thống thương mại điện tử bán giày dép", mục tiêu, đối tượng và phạm vi nghiên cứu của đồ án.
- Tìm hiểu tổng quan về kiến trúc phần mềm, mô hình 3 lớp MVC (Model - View - Controller).
- Nghiên cứu cơ sở lý thuyết về giao thức HTTP, các ngôn ngữ bổ trợ front-end bao gồm HTML, CSS, JavaScript và framework Bootstrap.

## 2. Tài liệu liên quan
- Các tài liệu, sách chuyên ngành về lập trình web hướng đối tượng.
- Khóa học lập trình hướng dẫn ASP.NET MVC.
- Mã nguồn giao diện Template sử dụng cho trang Dashboard quản trị.

## 3. Khó khăn gặp phải
- Hệ thống hóa lại kiến thức cốt lõi về mô hình MVC nhằm tách biệt rõ ràng các thành phần hoạt động: Model quản lý dữ liệu, View hiển thị giao diện và Controller xử lý yêu cầu.
- Gặp khó khăn trong việc nhúng và cấu hình các thư viện front-end tối ưu, chuẩn bị và thiết lập môi trường cài đặt .NET Framework kết hợp Visual Studio.

## 4. Kết quả đạt được
- Hoàn thành đề cương chi tiết và mục lục tổng quan của đồ án.
- Hoàn thiện Chương 1 (Nghiên cứu lý thuyết cơ sở) trong file báo cáo, định nghĩa rõ ràng các thuật ngữ và từ viết tắt liên quan.
